## Linguagem:
Java foi criada em 1991 por uma equipe da Sun Microsystems liderada por James Gosling, inicialmente com o nome Oak, para programar dispositivos eletrônicos como televisores.
O nome foi mudado para Java, inspirado no café da ilha de Java.
E uma linguagem de programação que atravessa décadas sem perder relevância. Com sua filosofia "escreva uma vez, execute em qualquer lugar", ela revolucionou o desenvolvimento de software ao permitir que aplicações rodem em diferentes sistemas operacionais sem precisar reescrever o código.

Ela foi lançada oficialmente em 1995, com o grande diferencial de ser portável: o código pode rodar em qualquer sistema com a JVM (Java Virtual Machine). Desde então, tornou-se uma das linguagens mais usadas no mundo, especialmente em sistemas corporativos, web e aplicativos Android.
